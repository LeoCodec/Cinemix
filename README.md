# 🎬 Cinemix

Aplicación Android para gestionar tu colección personal de películas, series y libros.

## ✨ Características

- 📽️ Registra películas, series y libros
- 🖼️ Sube portadas desde tu galería
- ⭐ Califica cada título (1-5 estrellas)
- ❤️ Marca tus favoritos
- 🔢 Lleva el conteo de cuántas veces lo viste/leíste
- 🔍 Filtra por tipo: Todos / Películas / Series / Libros
- ☁️ Datos sincronizados en Firebase Firestore

## 🛠️ Tecnologías

- Kotlin + Android Studio
- Firebase Firestore, Storage y Auth
- Navigation Component + Bottom Navigation
- ViewBinding + ViewModel + LiveData
- Glide para carga de imágenes
- Material Design 3
